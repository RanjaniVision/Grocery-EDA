# Grocery Chain Sales Dataset

## Overview

This dataset contains transactional records from multiple grocery store branches. It is designed for data analysis, visualization, business intelligence, and machine learning projects. The data captures customer purchases, store information, product details, pricing, discounts, and loyalty rewards.

The dataset can be used to explore customer purchasing behavior, sales performance, pricing strategies, and loyalty program effectiveness.

---

## Dataset Information

- **File Name:** `grocery_chain_data.csv`
- **Format:** CSV
- **Records:** Transaction-level data
- **Domain:** Retail / Grocery Sales

---

## Features

| Column | Description |
|----------|-------------|
| `customer_id` | Unique identifier for each customer |
| `store_name` | Name of the grocery store branch |
| `transaction_date` | Date of the purchase |
| `aisle` | Product category or aisle |
| `product_name` | Purchased product |
| `quantity` | Number of items purchased |
| `unit_price` | Price per unit |
| `total_amount` | Total amount before discounts |
| `discount_amount` | Discount applied to the transaction |
| `final_amount` | Final amount paid after discount |
| `loyalty_points` | Loyalty points earned by the customer |
